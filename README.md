# Faceroom

A tiny Mac app that puts a floating strip of live video in the corner of your screen.

## Download

**[⬇️ Download Faceroom.dmg](https://github.com/eliaspfeffer/faceroom-releases/releases/latest/download/Faceroom.dmg)**

1. Open the DMG → drag `Faceroom.app` to Applications
2. Launch from Applications → allow camera access when prompted

Updates are delivered automatically — once you have the app, you'll get every new version silently in the background.

## Requirements

macOS 13 or later
